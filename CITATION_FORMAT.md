# Справочник по формату цитирований v2.0

## Быстрое сравнение

| Аспект | Старый формат (v1.x) | Новый формат (v2.0) |
|--------|---------------------|---------------------|
| **Формат** | `[[TK_RB]]` | `ТК РБ - РАЗДЕЛ II. ГЛАВА 2 - Статья 16` |
| **Читаемость** | Низкая (ID не информативен) | Высокая (полный путь) |
| **Навигация** | Нет структуры | Раздел → Глава → Статья |
| **Поддержка** | ❌ Устарел | ✅ Активно поддерживается |

---

## Структура нового формата

```
[Аббревиатура] - [Раздел]. [Глава] - [Статья] [Пункт]
      ↓              ↓         ↓         ↓        ↓
   ТК РБ      -   РАЗДЕЛ II. ГЛАВА 2 - Статья 16  п1
```

### Компоненты:

1. **Аббревиатура** (обязательно)
   - Краткое название документа
   - Примеры: `ТК РБ`, `Декрет №8`, `Пост. №40`

2. **Раздел** (опционально)
   - Римскими цифрами: `РАЗДЕЛ II`, `РАЗДЕЛ III`
   - Используется для крупных законодательных актов

3. **Глава** (обязательно)
   - Арабскими цифрами: `ГЛАВА 2`, `ГЛАВА 10`
   - Присутствует во всех документах

4. **Статья** (обязательно)
   - Основная единица ссылки: `Статья 16`, `Статья 110`

5. **Пункт** (опционально)
   - Детализация: `п1`, `п2`, `п3`

## Правила форматирования

### ✅ Правильно:

```
ТК РБ - РАЗДЕЛ II. ГЛАВА 2 - Статья 16
Декрет №8 - ГЛАВА 1 - Статья 2 п1
ТК РБ - РАЗДЕЛ III. ГЛАВА 10 - Статья 111
Пост. №40 - п.10
```

### ❌ Неправильно:

```
ТК РБ - Статья 16                    (нет раздела и главы)
[[TK_RB]]                            (старый формат)
ТК РБ                                (нет конкретной статьи)
Статья 16 ТК РБ                      (неправильный порядок)
ТК РБ - раздел 2 - глава 2           (строчные буквы, арабские цифры для раздела)
```

## Интерактивность цитат

Цитаты в интерфейсе отображаются как **интерактивные бейджи**:

### Одинарный клик
- Открывает **предпросмотр** документа в модальном окне
- Показывает релевантный фрагмент (200-300 символов)
- Кнопка "Читать документ полностью"

### Двойной клик
- Открывает **полный документ** в модальном окне
- Автоматически **подсвечивает** релевантный текст
- Прокручивает к выделенному фрагменту

### Hover
- Показывает подсказку
- Текст: "Двойной клик — открыть документ с подсветкой этого фрагмента"

## Технические детали

### Регулярное выражение для парсинга

```javascript
// Паттерн для каждого источника
const pattern = new RegExp(
  escapedCitation + '\\s*-\\s*[^,\\.\\n]+',
  'gi'
);

// Пример для "ТК РБ":
/ТК РБ\s*-\s*[^,\.\n]+/gi
```

### Извлечение источников из текста

```javascript
function extractUsedSources(text, allSources) {
  const citedSources = new Set();

  allSources.forEach(source => {
    const escapedCitation = source.citation.replace(/[.*+?^${}()|[\]\\]/g, '\\$&');
    const pattern = new RegExp(escapedCitation + '\\s*-\\s*[^,\\.\\n]+', 'gi');

    if (text.match(pattern)) {
      citedSources.add(source.id);
    }
  });

  return allSources.filter(s => citedSources.has(s.id));
}
```

### Маппинг аббревиатур на источники

```javascript
// Из constants.ts
const MOCK_SOURCES = [
  {
    id: 'TK_RB',
    citation: 'ТК РБ',  // Используется для поиска

    title: 'Декрет Президента РБ №8 "О развитии цифровой экономики"',
    // ...
```

## Алгоритм поиска в документах

### Шаг 1: Извлечение контекста
```javascript
// Удаление императивных глаголов
"Подготовьте декларацию..." → "Декларацию..."

// Удаление всех цитат
"Текст ТК РБ - РАЗДЕЛ II..." → "Текст"

// Извлечение текущего пункта списка
"1. Текст\n2. Другой текст" → "Текст"
```

### Шаг 2: Фильтрация слов
```javascript
// Убираем служебные слова
const commonWords = ['для', 'это', 'как', 'или', 'при', 'том', 'что'];

// Убираем короткие слова (< 3 символов)
"декларацию о доходах" → ["декларацию", "доходах"]
```

### Шаг 3: Поиск совпадений
```javascript
// Подсчет совпадений
matchRatio = matchingWords / searchWords

// Порог успешного поиска: 30%
if (matchRatio >= 0.3) {
  highlightParagraph();
}
```

## FAQ

### Можно ли использовать несколько цитат в одном пункте?

Да, через запятую:

```
Текст с несколькими источниками
ТК РБ - РАЗДЕЛ II. ГЛАВА 2 - Статья 16, Декрет №8 - ГЛАВА 1 - Статья 2
```

### Что если документ не имеет разделов?

Опускаем раздел:

```
Декрет №8 - ГЛАВА 1 - Статья 2
Пост. №40 - п.10
```

### Регистр имеет значение?

Для поиска источников - **нет** (регистронезависимый).
Но рекомендуется соблюдать формат для читаемости:
- **РАЗДЕЛ**, **ГЛАВА** - заглавные
- **Статья** - с заглавной буквы

### Как добавить новый источник?

1. Добавьте в `constants.ts` в массив `MOCK_SOURCES`
2. Обязательные поля:
   - `id` - уникальный идентификатор
   - `citation` - аббревиатура для цитирования
   - `title` - полное название
   - `type` - тип документа (PDF/DOC/XLSX/WEB)
   - `fullContent` - полный текст для RAG

```javascript
{
  id: 'NEW_DOC',
  citation: 'НД',
  title: 'Новый документ',
  type: 'PDF',
  url: '#',
  fullContent: 'Полный текст документа...'
}
```
